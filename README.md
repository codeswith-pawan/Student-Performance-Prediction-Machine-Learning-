
# 🎓 Student Performance Prediction (ML + Streamlit)

## 🚀 Live Demo

🔗 https://ai-student-performance-checker.streamlit.app/

---

## 📌 Overview

This project is an **AI-powered web application** that predicts whether a student is likely to **PASS or FAIL** based on their academic scores.

It also provides:

* 📊 Real-time analytics
* ⚠️ Risk level detection
* 📈 Data visualization
* 📂 Batch processing via CSV

---

## 🧠 Features

* 🔹 **Single Prediction** (Enter scores manually)
* 🔹 **Batch Analysis** (Upload CSV file)
* 🔹 **AI Confidence Score**
* 🔹 **Risk Level Detection (Low / Medium / High)**
* 🔹 **Radar Chart Visualization**
* 🔹 **Custom Pass Threshold**
* 🔹 **Download Results**
* 🔹 **Clean & Interactive UI**


## 📊 How it works

* Takes **Reading, Writing, Math scores**
* Applies **Machine Learning model (Random Forest)**
* Predicts:

  * ✅ PASS
  * ❌ FAIL
* Shows:

  * Confidence %
  * Risk Level
  * Performance Insights
